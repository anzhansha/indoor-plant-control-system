# Project figures

The figures in this directory are curated exports from the diploma presentation and project archive. The main README places them in the corresponding data, identification, simulation, and hardware sections.

## Included

- `24h_PRBS_valve_signal.png` — initial 24-hour PRBS valve schedule.
- `arx_fit.png` — ARX validation comparison.
- `arx441_resid.png` — residual correlation diagnostics.
- `ss_fit.png` — three-state N4SID validation comparison.
- `try47_moisture_plot.png` — constrained 35-day moisture simulation.
- `x1_est_vs_true.png` — Kalman state-estimation comparison.
- `daily_water.png` — daily simulated water and pump ON time.
- `pump_durations.png` — despite its filename, this plot shows pump cycles per day.
- `raw_u.png` — unconstrained LQR-integral command.
- `hw_stack_photo_s.png` — conceptual hardware stack.
- `block_diagram_hw.png` — early prototype control flowchart.
- `field_photo1.png`, `field_photo2.png` — physical indoor lettuce setup.
- `designlqr.png`, `controlloop.png` — archived Python implementation excerpts.
- `control_logs_snippet1.png`, `control_logs_snippet2.png` — archived Raspberry Pi log snapshots.

`ss1_resid.png` was not copied because it is byte-identical to `arx441_resid.png`. Unrelated logo assets were also excluded.
