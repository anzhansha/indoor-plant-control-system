# Hardware scope

## Verified project role

- ESP32-S3: sensor acquisition and communication node.
- Soil-moisture sensor: baseline moisture measurement.
- Temperature/humidity sensor: environmental measurements.
- Valve or pump relay state: baseline irrigation record.
- ThingSpeak: remote collection of the baseline time series.

## Reported but not currently reproducible

- Raspberry Pi execution of a Python translation of the controller.
- Closed-loop physical controller testing.

No Raspberry Pi source, configuration, or logs are currently available. The repository therefore records these as reported project activities without publishing quantitative deployment claims.

Credentials, ThingSpeak API keys, Wi-Fi details, IP addresses, and private account identifiers must never be committed.
